Documentation

Learn how to integrate with the WhatsApp Manage API.

Authentication

All API requests must be authenticated using the API Key obtained from your active session.

Authorization: Bearer <YOUR_API_KEY>

Send Message

Endpoint to send a text message.

POST /api/send-message 
curl -X POST http://localhost:3000/api/send-message \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "to": "1234567890",
  "message": "Hello World!"
}'

Send Image

Endpoint to send an image with an optional caption.

POST /api/send-image 
curl -X POST http://localhost:3000/api/send-image \
-H "Authorization: Bearer YOUR_API_KEY" \
-H "Content-Type: application/json" \
-d '{
  "to": "1234567890",
  "url": "https://example.com/image.jpg",
  "caption": "Check this out!"
}'

Webhooks

Incoming messages and status updates will be sent to the webhook URL configured on your session via POST request.
