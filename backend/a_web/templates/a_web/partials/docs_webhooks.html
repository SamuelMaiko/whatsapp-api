Webhooks

Incoming messages and status updates will be sent to the webhook URL configured on your session via POST request.

COPY
{
  "sessionId": "sess_123...",
  "phoneNumber": "254700000000",
  "pushName": "John Doe",
  "text": "Hello!",
  "imageUrl": null
}

The following events trigger a webhook notification:

  • New text messages received
  • New image messages received (with URL)
  • Session status changes (QR, Connected, Disconnected)
